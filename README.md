# Hybrid OS v6

GitHub Pages + Supabase training app.

## Update steps
1. Run `supabase-v6.sql` in Supabase SQL Editor.
2. Replace `SUPABASE_ANON_KEY` in `app.js` with your anon public key.
3. Upload `index.html`, `style.css`, `app.js` to GitHub Pages repo root.
4. Commit and refresh your GitHub Pages app.
