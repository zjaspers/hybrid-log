# Hybrid OS v9

Hybrid Athlete Operating System V2.0 app.

Adds:
- V2.0 mission and non-negotiables
- 15–20 minute Zone 2 after lift days
- 20–30 minute standalone Zone 2 runs
- direct biceps/triceps/shoulder work programmed every lifting day
- upper chest emphasis
- Sunday check-in fields for arm measurement, Zone 2 pace, and photo notes
- better progression logic for accessories versus main compounds
- copy set 1 to all sets retained

## Install
1. Run `supabase-v9.sql` in Supabase SQL Editor.
2. Replace GitHub files with `index.html`, `style.css`, and `app.js`.
3. Commit and refresh GitHub Pages.
